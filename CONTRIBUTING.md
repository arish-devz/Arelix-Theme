# Contributing to Arelix Theme

Thank you for considering contributing to the Arelix Theme!

## How to Contribute
1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Make your changes and commit them with descriptive messages.
4. Push your changes to your fork.
5. Submit a pull request to the main repository.

## Reporting Issues
If you encounter any issues, please open a bug report on GitHub with detailed steps to reproduce.

## Security Vulnerabilities
If you discover a security vulnerability, please report it via privacy email or discord instead of opening a public issue.
